FAQ

Q: What is this?

A: This site helps you to export your PoliMi™ lecture timetable in the Google Calendar App, creating an event for each lecture. The events are set with a weekly recurrence period so, in case your professor changes the lectures hours, you can easily edit all the info for that lecture from the Google App.

Q: Do I need a Google account?

A: Yes. The popup that appears after hitting the "Authorize" button will give an option to create an account if you don't have one.

Q: Where can I find the timetables?

A: Go to the Polimi Online Services, find the Lecture Timetables link and then you'll have to select the Text Timetable tab. NOTE: this will probably take long time to do from the mobile version of the Online Services, so I strongly suggest to use the Desktop version.

Q: What does the create new calendar option mean?

A: Google Calendar gives you a primary calendar and the possibility to add other secondary ones. By default, this site creates a new calendar named "PoliMi" and puts the events there, but if you prefer to have a single calendar uncheck that option and you'll find all of your lectures in the primary calendar. Note that you can still move the events between your secondary and primary calendars, so this is not a binding option anyway.

Q: How can I contribute?

A: Hey, thanks! This site is quite simple and I don't think it needs any particular attention, but if you have any suggestion or a bug to report you can open a new Issue on GitHub. All the code is accessible from there. Pull requests are also very appreciated, in case you find any typo or want to contribute to the site in any way (for example, by providing an Italian translation).

A: That's not a question.
